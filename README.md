# تشغيل تطبيق Streamlit

اتبِع الخطوات التالية لتثبيت المتطلبات وتشغيل التطبيق محليًّا:

## 1) تثبيت الاعتمادات
- أنشئ بيئة افتراضية (اختياري لكن مفضّل):
  ```bash
  python -m venv .venv && source .venv/bin/activate
  ```
- ثبّت الحزم من خلال `pyproject.toml`:
  ```bash
  pip install -e .
  ```

> التطبيق يضبط اتصال قاعدة البيانات تلقائيًّا إلى SQLite في الذاكرة إذا لم يتم ضبط المتغير `DATABASE_URL`.

## 2) تشغيل التطبيق
- من جذر المشروع نفّذ:
  ```bash
  streamlit run "app (1).py" --server.address 0.0.0.0 --server.port 8501
  ```
- بعد التشغيل ستحصل على رابط مباشر مثل: http://localhost:8501 أو العنوان الظاهر في الطرفية.

## 3) تخصيص قاعدة البيانات (اختياري)
إذا أردت استخدام قاعدة بيانات خارجية، عيّن المتغير `DATABASE_URL` قبل التشغيل، مثلاً:
```bash
export DATABASE_URL="postgresql://user:pass@host:5432/dbname"
```

## 4) إيقاف التطبيق
أغلق الطرفية أو اضغط `Ctrl + C` في نافذة التشغيل.
